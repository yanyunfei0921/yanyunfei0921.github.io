第四章、当索引行不通时

字典是python中唯一的映射数据结构。值与键相对应。

4.1 字典的用途

用列表储存一个电话簿，需要一个列表记录人名，一个列表记录电话号码。查询时也得动点脑筋。

>>> name=["Yan","Li","Zhao"]
>>> tel=["1303244321","123456","97580"]
>>> tel[name.index("Li")]
'123456'

其实我们更希望直接建立一个人名与电话号码的映射，而不是通过索引这个桥梁。

4.2 创建和使用字典

字典实现的电话簿如下：

>>> phonebook={"Yan":"1093244321","Li":"123456","Zhao":"97580"}
>>> phonebook["Li"]
'123456'

字典的每个item由键和值组成，一个字典中键不能重复，值可以。这也很好理解，毕竟访问时是以键为索引。

4.2.1 函数dict

函数dict可使用其他映射创建字典。

>>> tup1=("name","Li")
>>> tup2=("age",18)
>>> items=[tup1,tup2]
>>> dict(items)
{'name': 'Li', 'age': 18}

也可使用关键字参数：

>>> dict(name="Li",age:18)
{'name': 'Li', 'age': 18}

4.2.2 基本的字典操作

字典与列表不同之处：

  1. 字典的键必须为不可变数据类型。
  2. 字典可对不存在的键赋值，此时会自动创建。
  3. 字典通过键判断是否存在该item，列表通过值。

基本行为：
| len(dir)|返回字典的item数|
|dir[k]|返回键为k的值|
|dir[k]=v|将键k对应的值赋为v|
|del dir[k]|删除该item|
|k in dir|检查键为k的item是否在dir中|

4.2.3 通过字典指定字符串格式设置
>>> dir={"name":"Li","age":18}
>>> "{name} is {age}".format_map(dir)
'Li is 18'
4.2.4 字典方法
  1. clear
    删除字典中所有元素。注意clear与赋值为空字典的区别。
    >>> dir1={}
>>> dir2=dir1
>>> dir1["name"]="Li"
>>> dir2
{'name': 'Li'}
>>> dir1.clear()
>>> dir2
{}
    >>> dir1={}
>>> dir2=dir1
>>> dir1["name"]="Li"
>>> dir2
{'name': 'Li'}
>>> dir1={}
>>> dir2
{'name': 'Li'}

对比发现，clear清空了dir1指向的内存，而直接进行赋值操作改变了dir1的指向，它不再与dir2指向同一位置，而是直接指向None。

  1. copy
    从上述对于clear的阐述中就可以看出，与列表一样，直接赋值是不靠谱的。字典同样存在浅拷贝、深拷贝的问题。copy方法就是一种浅拷贝。如果字典的值都是常量，这样的浅拷贝没问题。但如果字典的值为字典或列表等，就会出问题。

此时可采用deepcopy方法。

  1. fromkeys
    创建一个指定键，值都相同的字典，值默认为空。
    >>> dict.fromkeys(["name","age"],"unknown")
{'name': 'unknown', 'age': 'unknown'}
  2. get
    直接通过键索引时，如果键不存在，就会出现错误。但使用get方法访问不存在的键时会返回None，提供了较为宽松的访问。
  3. items
    返回一个字典视图类型的数据。
    >>> dir={'name': 'unknown', 'age': 'unknown'}
>>> dir.items()
dict_items([('name', 'unknown'), ('age', 'unknown')])
  4. keys
    返回一个仅包含键的字典视图。
    >>> dir={'name': 'unknown', 'age': 'unknown'}
>>> dir.keys()
dict_keys(['name', 'age'])
  5. pop
    出栈指定键对应的值。
    >>> dir={'name': 'unknown', 'age': 'unknown'}
>>> dir.pop("age")
'unknown'
>>> dir
{'name': 'unknown'}
  6. popitem
    随机出栈一个item。
  7. setdefault
    与get类似，只不过在不存在时自动添加上所访问的不存在键值对。
    >>> dir={"name":"Li"}
>>> dir.setdefault("name","Wang")
'Li'
>>> dir
{'name': 'Li'}
>>> dir.setdefault("age",18)
18
>>> dir
{'name': 'Li', 'age': 18}
  8. update
    使用一个字典或其他映射或关键字参数来更新该字典。
  9. values
    返回一个只有值的字典视图。
